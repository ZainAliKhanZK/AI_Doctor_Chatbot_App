# 🤖 AI Assisted Doctor Chat App (Streamlit + LLM API)

A simple and interactive AI-powered chat application built using **Streamlit** and **Hugging Face / OpenAI-compatible LLM API**.  
The app allows users to enter prompts and receive real-time responses from a large language model through a clean web interface.

---

## 🚀 Features

- 💬 Chat with an AI assistant in real time  
- 🎯 Simple and user-friendly Streamlit UI  
- ⚡ Fast responses using LLM API integration  
- 🔐 Secure API key handling using environment variables  
- 🧠 System prompt-based assistant behavior control  

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit 🎈  
- OpenAI / Hugging Face Inference API  
- python-dotenv
